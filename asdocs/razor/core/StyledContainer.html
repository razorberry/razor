razor_internal function addBlueprint(tagOrClass:*, initializers:Object = null, depth:Number, controlFactory:ControlFactory = null):DisplayObject

Add a child component to this container.

Internal method for adding a razor component to this container. If the new component is a StyledContainer, this method ensures that the component is initialized in the correct order.

Parameters

tagOrClass:* — The skin ID or class of the component to add.
 
initializers:Object (default = null) — (optional) Any initial parameters can be pushed in to the component here.
 
depth:Number (default = NaN) — (optional) The depth at which to add the component
 
controlFactory:ControlFactory (default = null) — (optional) The controlFactory instance to use.

Returns
DisplayObject — The new component.

Example
In order to use this method, you need to import and use the razor_internal namespace. 
        import razor.core.razor_internal;  // Import the namespace
        ...
        use namespace razor_internal;  // Use the namespace.. this can be placed on the class level or inside your method.
        myLabel = addBlueprint(Label) as Label;  // Create the label and cast the result.
addStates()method 
protected function addStates(tag:String, depth:int, states:Array):DisplayObject

Add states for a particular child component of this container.

This allows you to automatically switch between states (for example, up/over/down states in a button) For an example, see the Button component.

You normally define states in the StyleSheets, and the corresponding component is created when the state is set (with setState())

Parameters

tag:String — The skin ID of the container
 
depth:int — The depth at which to create the container
 
states:Array — An array of state identifiers (strings).

Returns
DisplayObject — The current element for the state that was passed.

See also

getStateElement
setState

Example 
        myIcon = addStates("MyStatefulIcon", 1, ["Selected", "Unselected"]); // Define the states
        setState("Unselected");  // This will create MyStatefulIcon$Unselected in the Skin definition
        var myIcon:Container = getStateElement("MyStatefulIcon"); // Gets the current icon
        myIcon.move(2,2);

getStateElement()method 
protected function getStateElement(tag:String):DisplayObject

Get the component for a skin ID in its current state.

You would only use this if you have added components using addStates. For example, if you have added a button in three states, and the current state is the first state. Then this method would return the button in the first state. See the Button control for a more detailed example.

Parameters

tag:String — The skin ID of the component to retrieve

Returns
DisplayObject — The current component for the state.
